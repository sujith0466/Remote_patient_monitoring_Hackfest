# CareWatch (Frontend)

Frontend skeleton for the CareWatch hackathon project.

Tech stack: React + Vite, Tailwind CSS, Chart.js

Quick start:

1. cd frontend
2. npm install
3. npm run dev

Notes:
- Tailwind config and PostCSS config are included.
- This skeleton provides `Nurse` and `Doctor` dashboards and simple components.
- Follow the project plan: Frontend first, then Backend, then Alerts.
